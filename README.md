# Front End Web

Frontend web responsive cho he thong `ecommerce-anti-fake`, tach thanh repo rieng va chay voi backend gateway.

## Stack

- React 18
- TypeScript
- Vite
- React Router

## Features hien tai

- Dang nhap / dang ky
- App shell responsive cho desktop va mobile
- Dashboard tong quan
- User profile va KYC
- Shops workspace
- Products / offers workspace
- Distribution workspace
- Orders workspace
- Affiliate workspace
- Admin dashboard workspace

## Cai dat

```bash
npm install
```

## Chay local

```bash
cp .env.example .env
npm run dev
```

Mac dinh frontend se goi:

- `VITE_API_BASE_URL=http://localhost:3001`

Can dam bao `api-gateway` cua backend dang chay truoc.

## Build

```bash
npm run build
```
